Assessment rules and declaration

Read this before you start

  • You may take the assessment for this role once every 1 month.
  • The assessment is 1 hour long.
  • Each role has a different assessment question set.
  • If you leave the assessment, switch tabs, refresh, or exit fullscreen, you will be given a zero.
  • If you submit without answering, you will be given a zero.
The assessment will enter fullscreen before questions are shown.
